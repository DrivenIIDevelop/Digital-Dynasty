## Setup

```sh
# clone repo
git clone <repo>
cd <repo>
yarn # install packages
yarn dev # dev serve
```

## Tech Stack

- Package Manager: Yarn
- Frontend: ReactJS
- Styling: Tailwind
- Formatter: Prettier
- Backend: ExpressJS
- Database: MongoDB
- Production: Docker-compose
